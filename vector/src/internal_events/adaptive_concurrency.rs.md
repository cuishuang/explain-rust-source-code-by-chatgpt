# File: vector/src/internal_events/adaptive_concurrency.rs

AdaptiveConcurrencyLimitData也提供了一些方法来进行并发级别的调整。它们会根据当前的负载情况和配置项来动态地增加或降低并发限制级别。这些方法的目标是通过自动调整并发级别来实现更高的性能和资源利用率。
