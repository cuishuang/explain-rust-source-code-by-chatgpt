# File: rayon/rayon-core/src/compile_fail/rc_return.rs

在rc_return.rs文件中，包含了一系列Rust代码片段，这些代码片段在理论上是无法通过编译的。通过测试这些编译失败的情况，可以确保rayon库在编译时能够正确地处理错误。
